Installation Guide

Windows Setup

  1. Download the Magic.Tracker.exe file.
  2. Double-click to run. If "Windows protected your PC" appears, click More info then Run anyway.
  3. If Windows Firewall asks, allow access for Private Networks (Home Wi-Fi).
  4. The app will open and show a 4-digit PIN.

macOS Setup

  1. Download and unzip Mac_App_MagicTracker.zip.
  2. Drag Magic Tracker.app into your Applications folder.
  3. Important Security Step: Open Terminal and paste this command to fix "Damaged/Unverified" error:
    sudo xattr -rd com.apple.quarantine /Applications/"Magic Tracker.app"
  4. Open the app. Go to System Settings > Privacy & Security > Accessibility and allow Magic Tracker.
