Installation Guide

Windows Setup

  1. Download the Windows_App_MagicTracker.zip file.
  2. Extract (Unzip) the folder. Do not run from inside the zip.
  3. Double-click Magic Tracker.exe.
  4. If a blue box says "Windows protected your PC", click More info then Run anyway.

macOS Setup

  1. Download and unzip Mac_App_MagicTracker.zip.
  2. Drag Magic Tracker.app into your Applications folder. (Mandatory)
  3. Fix "App is Damaged" Error (Required):
    • Open Terminal (Cmd + Space, search Terminal).
    • Copy and paste this command exactly:
      sudo xattr -rd com.apple.quarantine "/Applications/Magic Tracker.app"
    • Press Enter. Type your Mac password (it will be invisible) and press Enter again.
  4. Go to System Settings > Privacy & Security > Accessibility. Remove any old entry and add Magic Tracker again.
